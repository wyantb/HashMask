About HashMask

This browser extension is basically a wrapper around someone else's work - which can be found here - by Chris Dary. So lots of credit goes to him.

The extension part is us, a few guys from the Society of Software Engineers at RIT. If you have any problems, please contact any of the three developers listed below.

Developers:

  • Daniel Wallach (Project Lead) - dywallach@dywallach.net
  • Brian Wyant - wyantb@gmail.com
  • Nicolas McCurdy - thenickperson.com

You can also file bugs right here! Any notes of visual defects are welcome.

HashMask lives on GitHub and is released under the BSD license.

Cool projects are always built on the work of others. In our case, that includes Bootstrap by @mdo and @fat, jQuery, the Standard Javascript Crypto Library, and a few existing hash algorithms (links can be found in the help section for each).

So, we hope you like HashMask. Thanks for checking it out!
